Introducción

La transformación digital de las organizaciones ha impulsado la adopción de tecnologías que permitan una gestión más eficiente, flexible y escalable de los recursos informáticos. En este contexto, la virtualización, la contenerización y la orquestación de servicios se han convertido en herramientas fundamentales para el desarrollo y despliegue de soluciones tecnológicas modernas. El presente proyecto tiene como finalidad construir y documentar una infraestructura tecnológica básica mediante la integración de máquinas virtuales con sistema operativo Linux, contenedores Docker para la implementación de servicios esenciales y Kubernetes Minikube para la administración y despliegue de aplicaciones. A través de estas tecnologías se busca simular un entorno de trabajo similar al utilizado en escenarios profesionales, permitiendo comprender el ciclo completo desde la creación de un entorno virtualizado hasta la puesta en marcha de servicios contenerizados y su posterior orquestación. Durante el desarrollo del proyecto se realizaron actividades relacionadas con la instalación y configuración de sistemas Linux en entornos virtuales, la implementación de servicios utilizando Docker Engine y Docker Compose, y el despliegue de aplicaciones mediante Kubernetes Minikube. Cada una de estas etapas fue documentada mediante evidencias técnicas que permiten demostrar el proceso realizado, las configuraciones aplicadas y los resultados obtenidos. Además de fortalecer los conocimientos técnicos, este proyecto fomenta habilidades de trabajo en equipo, análisis de problemas, documentación técnica y sustentación de decisiones tecnológicas. La elaboración de un sitio web para la publicación de las evidencias busca garantizar que toda la información del proyecto se encuentre organizada, accesible y disponible para su consulta, facilitando la comprensión de cada una de las actividades desarrolladas.
